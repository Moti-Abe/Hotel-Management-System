# Hotel Management System

**Derartu Hotel Management System** is a Java-based desktop application designed to streamline hotel operations by providing a centralized platform for managing rooms, guests, reservations, billing, and related tasks.

This project was developed as a complete hotel management solution that can be used by hotel staff to efficiently handle daily workflows and maintain organized records.
